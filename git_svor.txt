2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone - The git clone command copies an existing Git repository
	git log - git log sýnir þér öll commits sem þú hefur gert
	git status sýnir hvaða branchi ég er í
	git diff -  sýnir breytingar á commits 
	git checkout - skitpir um branch

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
Working directory er staðurinn þar sem öll skjölin eru geymd.

Staging area er þar sem þú ert að vinna áður en þú gerir commmit.

Repository þar geymist allt history.
